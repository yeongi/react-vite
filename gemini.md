# Project Context

## Overview
This is a frontend web application built with **React** and **TypeScript**, using **Vite** as the build tool. It appears to be initialized from a standard Vite template.

## Tech Stack
- **Framework:** React 18
- **Language:** TypeScript 5
- **Build Tool:** Vite 5
- **Linting:** ESLint

## Scripts
(Run via `npm run <script>`)
- `dev`: Starts the development server.
- `build`: Type-checks (`tsc`) and builds the application for production.
- `lint`: Runs ESLint to check for code quality.
- `preview`: Serves the built application locally for preview.

## Key Files & Directories
- `src/main.tsx`: Application entry point. Mounts the React app.
- `src/App.tsx`: Root component.
- `vite.config.ts`: Vite configuration file.
- `tsconfig.json`: Main TypeScript configuration.
